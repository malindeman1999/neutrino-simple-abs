Detector Goal

This design specifies a thermal kinetic inductance detector (TKID) operated as a calorimeter for neutrino mass measurement with low-Q isotopes such as Ho-163.

Thermal regime chain: \(E \rightarrow \text{absorber} \rightarrow \Delta T \rightarrow \text{KID frequency shift}\).

Pixel Architecture

Each pixel contains:

  • Absorber (includes radioactive isotope)
  • Membrane (thermal isolation)
  • KID resonator (thermometer)

Thermal topology: \(\text{Absorber} \leftrightarrow \text{KID} \leftrightarrow \text{Membrane} \leftrightarrow \text{Bath}\)

Core Design Rules

  1. \(\tau_{qp} \ll \tau_{res} \ll \tau_{th}\)
  2. \(\tau_{res} \le \tau_{th}/3\)
  3. Low-frequency phonon phase noise \(>\) obtainable TLS phase noise at 100 Hz offset
  4. \(P_0 < P_{\mathrm{bif}}\) (hard limit)
  5. \(P_0/P_{\mathrm{bif}} > 0.5\) (Rule 5 soft floor)
  6. \(P_{\mathrm{bif,dBm}}\) in typical 100 mK, ~1 GHz MKID range (Rule 6)
  7. \(\Delta T_{\mathrm{event}} > 1\,\mathrm{mK}\) (Rule 7)
  8. \(\Delta T_{\mathrm{event}} < 100\,\mathrm{mK}\) (Rule 8)
  9. \(\sigma_{E,\mathrm{th,eV}} < 0.1\,\mathrm{eV}\) (Rule 9)
  10. Operate near bifurcation from below as a soft optimization goal
  11. \(G_{AK} \gg G_{AB}\)
  12. Maintain GHz resonator frequency
  13. Minimize TLS via capacitor design
  14. Keep heat capacity minimal
  15. Ensure phonon containment

Core Rule Checks

Rule 1 (\(\tau_{qp}\ll\tau_{res}\ll\tau_{th}\)) is checked by the ratios:

\[ \frac{\tau_{qp}}{\tau_{res}},\qquad \frac{\tau_{res}}{\tau_{th}}. \]

Current code flags Rule 1 pass when both are below 0.1.

Rule 2 (\(\tau_{res}\le\tau_{th}/3\)) is checked by:

\[ \frac{\tau_{res}}{\tau_{th}/3}\le 1. \]

Rule 3 (margin criterion) requires low-frequency phonon phase ASD to exceed modeled TLS phase ASD at 100 Hz offset:

\[ \sqrt{S_{\phi,\mathrm{ph}}}^{\,\mathrm{simple}}>\sqrt{S_{\phi,\mathrm{TLS}}}(100\,\mathrm{Hz}). \]

Rule 5 applies a soft floor on operating fraction:

\[ \frac{P_0}{P_{\mathrm{bif}}}>0.5. \]

Rule 6 checks that \(P_{\mathrm{bif,dBm}}\) lies in the chosen typical range:

\[ P_{\mathrm{bif,min}}^{\mathrm{typ}} \le P_{\mathrm{bif,dBm}} \le P_{\mathrm{bif,max}}^{\mathrm{typ}}. \]

Rule 7 requires a resolvable event temperature rise:

\[ \Delta T_{\mathrm{event}} > 1\,\mathrm{mK}. \]

Rule 8 caps event temperature rise to stay in a small-signal regime:

\[ \Delta T_{\mathrm{event}} < 100\,\mathrm{mK}. \]

Rule 9 limits the thermal fluctuation energy scale:

\[ \sigma_{E,\mathrm{th,eV}} < \sigma_{E,\mathrm{target}}, \quad \sigma_{E,\mathrm{target}}=0.1\,\mathrm{eV}\ \text{(default)}. \]

Soft goal: choose \(P_0/P_{\mathrm{bif}}\) close to 1 from below.

Source Activity Formulas

Absorber volume from count rate and Ho activity:

\[ V_{\mathrm{abs}}=\frac{R}{A_{\mathrm{Ho}}}. \]

Cubic absorber edge and dimensions:

\[ a_{\mathrm{abs}}=V_{\mathrm{abs}}^{1/3},\qquad L_{\mathrm{abs}}=W_{\mathrm{abs}}=t_{\mathrm{abs}}=a_{\mathrm{abs}}. \]

Python: Sensor.absorber_volume_m3, Sensor.absorber_edge_m.

Membrane Geometry Formulas

Membrane dimensions from absorber, KID, and margin:

\[ L_{\mathrm{mem}}=L_{\mathrm{abs}}+L_{\mathrm{KID}}+\Delta_{\mathrm{mem}},\qquad W_{\mathrm{mem}}=W_{\mathrm{abs}}+W_{\mathrm{KID}}+\Delta_{\mathrm{mem}}. \]

Characteristic span:

\[ S_{\mathrm{mem}}=\max(L_{\mathrm{mem}},W_{\mathrm{mem}}). \]
