From M(\omega) to M_t

Following the resonator-bolometer paper (Eq. 14 discussion), define:

\[ D_0 = M(0),\qquad D_1 = -i\,\frac{\partial M}{\partial \omega}. \]

The linear time-domain dynamics matrix is then:

\[ M_t = -D_1^{-1}D_0. \]

Paper in this repo: 2014 resonator-bolometer theory PDF.

Python: Sensor.d0_matrix, Sensor.d1_matrix, Sensor.mt_matrix.

Stability Criterion

Let \(\lambda_i\) be eigenvalues of \(M_t\). The system is stable if all real parts are negative:

\[ \Re[\lambda_i(M_t)] < 0\quad \forall i. \]

Reported outputs from code:

  • \(\max\Re[\lambda(M_t)]\)
  • Boolean pass flag \(\mathbb{1}_{\mathrm{stable}}\)

Python: Sensor.mt_eigenvalues, Sensor.mt_max_real_part, Sensor.mt_stable.

Pulse Shortening Metric

If the system is stable, define the pulse-shortening ratio as thermal \(G/C\) normalized to the slowest modal rate:

\[ \rho_{\mathrm{short}}=\frac{G/C}{\min_i|\lambda_i(M_t)|}. \]

This is reported only when \(\Re[\lambda_i(M_t)]<0\ \forall i\). Values below 1 indicate shortening.

Python: Sensor.mt_pulse_shortening_ratio.
