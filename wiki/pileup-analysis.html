Assumptions

  • Average event rate: \(R\) from source activity and absorber volume
  • NEP sufficiency percent input: \(p_{\mathrm{NEP,suff}}\) (default \(10\%\))
  • Phase-total NEP spectrum integrated from high to low frequency
  • Coincidence veto window set by NEP-sufficient time \(\tau_{\mathrm{NEP,suff}}\)

The cumulative and full NEP-based energy resolutions are:

\[ \sigma_E^{\mathrm{hi\to lo}}(f)=\left[\int_{f}^{f_{\max}}\frac{4}{\mathrm{NEP}(f')^2}\,df'\right]^{-1/2},\qquad \sigma_{E,\mathrm{full}}=\left[\int_{f_{\min}}^{f_{\max}}\frac{4}{\mathrm{NEP}(f')^2}\,df'\right]^{-1/2}. \]

Define the first sufficient frequency:

\[ f_{\mathrm{NEP,suff}}=\max\left\{f:\sigma_E^{\mathrm{hi\to lo}}(f)\le\left(1+\frac{p_{\mathrm{NEP,suff}}}{100}\right)\sigma_{E,\mathrm{full}}\right\}, \quad \tau_{\mathrm{NEP,suff}}=\frac{1}{f_{\mathrm{NEP,suff}}}. \]

Any pulse with another arrival inside \(\pm\tau_{\mathrm{NEP,suff}}\) is rejected (both colliding pulses rejected). For Poisson arrivals, the per-event rejection probability is:

\[ P_{\mathrm{pileup,reject}}=1-e^{-2R\tau_{\mathrm{NEP,suff}}}. \]

Model Outputs

See the current computed values in Python Estimates:

Default Setting

The default model setting is \(p_{\mathrm{NEP,suff}}=10\%\), meaning \(\tau_{\mathrm{NEP,suff}}\) is taken where cumulative NEP resolution comes within 10% while integrating from high to low of the full-band NEP resolution.
