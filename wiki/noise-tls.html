TLS Noise Vector

TLS resonance-frequency noise represented in the phase-like source direction.

Physical Expression

Common fractional frequency-noise form:

\[S_{TLS}(f)=A_{TLS}f^{-\beta}.\]

In the Lindeman mapping, TLS is treated as an effective fluctuating resonance frequency, aligned with the phase-like direction.

Design Scaling and Geometry Controls

For this project, TLS is parameterized directly by phase ASD at 100 Hz offset and a spectral exponent:

\[ \sqrt{S_{\phi,\mathrm{TLS}}}(100\,\mathrm{Hz}),\qquad \beta_{\mathrm{TLS}}. \]

Offset-frequency scaling is:

\[ \sqrt{S_{\phi,\mathrm{TLS}}}(\nu)=\sqrt{S_{\phi,\mathrm{TLS}}}(100\,\mathrm{Hz})\left(\frac{\nu}{100\,\mathrm{Hz}}\right)^{-\beta_{\mathrm{TLS}}/2}. \]

Then convert to fractional-frequency noise using the local phase slope:

\[ \left(\frac{S_f}{f_0^2}\right)_{\mathrm{TLS}}(\nu)= \left(\frac{\sqrt{S_{\phi,\mathrm{TLS}}}(\nu)}{f_0\left|d\phi/df\right|_{\delta f}}\right)^2. \]

IDC design intent is to reduce \(F\) and local electric field. For fixed stored energy \(U\),

\[ U=\frac{1}{2}CV^2\Rightarrow V=\sqrt{\frac{2U}{C}}, \]

so larger capacitance lowers field scale and typically reduces TLS coupling.

Lower \(f_0\) often helps indirectly because it usually implies larger \(C\) for fixed \(L\):

\[ f_0=\frac{1}{2\pi\sqrt{LC}},\qquad V\propto C^{-1/2}. \]

For first-pass quantitative estimates, see TLS Estimate. Geometry guidance is summarized in IDC Geometry.

Noise Vector Derivation

Starting from Eq. (18) phase-like Johnson vector as a basis direction, define TLS source as:

\[N_{TLS}(f)=M_{TLS}(f)\,N_{J,\phi}.\]

Define the multiplier from fractional-frequency-noise ratio:

\[ M_{TLS}(f)= \sqrt{ \frac{\left(S_f/f_0^2\right)_{TLS}(f)} {\left(S_f/f_0^2\right)_{J}(f)} }. \]

Using the project TLS parameterization:

\[ \left(\frac{S_f}{f_0^2}\right)_{\mathrm{TLS}}(\nu)= \left(\frac{ \sqrt{S_{\phi,\mathrm{TLS}}}(100\,\mathrm{Hz})\left(\dfrac{\nu}{100\,\mathrm{Hz}}\right)^{-\beta_{\mathrm{TLS}}/2} }{ f_0\left|d\phi/df\right|_{\delta f} }\right)^2. \]

For Johnson phase-noise PSD, use the uncorrelated quadrature sum from the Johnson page:

\[ S_{\phi,J}(f)=\left|[Y_{J,A}(f)]_{\phi}\right|^2+\left|[Y_{J,\phi}(f)]_{\phi}\right|^2. \]

Then use the general (detuning-capable) conversion:

\[ \left(\frac{S_f}{f_0^2}\right)_{J}(f,\delta f)= \frac{S_{\phi,J}(f,\delta f)}{f_0^2\,\left|d\phi/df\right|_{\delta f}^{\,2}}. \]

So \(M_{TLS}\) becomes

\[ M_{TLS}= \sqrt{ \frac{ \left(\dfrac{ \sqrt{S_{\phi,\mathrm{TLS}}}(100\,\mathrm{Hz}) \left(\dfrac{\nu}{100\,\mathrm{Hz}}\right)^{-\beta_{\mathrm{TLS}}/2} }{f_0\left|d\phi/df\right|_{\delta f}}\right)^2 }{ \left(\left|[Y_{J,A}]_{\phi}\right|^2+\left|[Y_{J,\phi}]_{\phi}\right|^2\right)\Big/\!\left(f_0^2\,\left|d\phi/df\right|_{\delta f}^{\,2}\right) } }. \]

Using \(N_{J,\phi}\) from the Johnson page, the explicit TLS vector is:

\[ N_{TLS}(f)= \begin{bmatrix} 0\\ i\,M_{TLS}(f)\sqrt{16k_BT_0/P_0}\\ 0 \end{bmatrix}. \]

Vector Component Units

\(N_{TLS}\) is also a mixed-unit \((r,\phi,T)\) vector, with component units:

\[ \mathrm{units}(N_{TLS})= \begin{bmatrix} 0\\ 1/\sqrt{\mathrm{Hz}}\\ 0 \end{bmatrix}. \]

Because \(N_{TLS}=M_{TLS}N_{J,\phi}\), the units follow directly from \(N_{J,\phi}\).

Use In Total Budget

Compute \\(Y_{TLS}=M^{-1}N_{TLS}\\), then combine in quadrature with other sources (and include correlations if measured).

Project criterion: in-band TLS phase noise should remain below the phonon-noise-equivalent floor over the analysis band.
