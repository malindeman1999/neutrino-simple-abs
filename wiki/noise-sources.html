How To Use This Section

Each page below gives (1) the physical noise expression, (2) mapping to source vector components, and (3) the normalized vector used in the resonator-bolometer model.

All sources are propagated using \\(Y=M^{-1}N\\), then converted to phase-direction NEP using Eq. (20):

\[ \mathrm{NEP}_{\phi}(\omega)=\left|\frac{[Y_{\mathrm{RMS}}(\omega)]_{\phi}}{(M^{-1})_{\phi,\,P}}\right|. \]

For matrix definition and implementation details, see M Matrix.
