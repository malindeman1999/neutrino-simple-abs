Paper Basis and Scope

This page now follows the resonator-bolometer model in M. A. Lindeman, J. Appl. Phys. 116, 024506 (2014), with emphasis on Eq. (4) and Eq. (10)-(20).

Model objective for this project: compute the measured quadrature noise, convert it to phase-direction noise, and then convert that to calorimeter NEP using the bolometer responsivity.

Nominal calculation setting for this wiki: \(f_0=1.0\,\mathrm{GHz}\) unless a section explicitly overrides it.

State and Measurement Vectors

Use the paper’s normalized demodulated state vector:

\[ Y(\omega)= \begin{bmatrix} r(\omega)\\ \phi(\omega)\\ T_{TB}(\omega) \end{bmatrix}, \]

and source vector \(N(\omega)\), related by Eq. (11):

\[ M(\omega)Y(\omega)=N(\omega). \]

Linearized Forward Model

The theory used here follows the paper’s sideband-to-demodulated linear system and normalized matrix form (Eq. (8)–(13)) rather than an external factorized \(R/H\) state-space model.

For implementation details of the matrix coefficients, see M Matrix.

Eq. (4): Sensitivities \( \alpha \) and \( \beta \)

In the paper, the complex sensitivities are defined by impedance derivatives (with \(I_A\) the bias-current amplitude):

\[ \alpha \equiv \frac{T}{R}\frac{\partial Z}{\partial T},\qquad \beta \equiv \frac{I_A}{R}\frac{\partial Z}{\partial I_A}. \]

Eq. (4) then gives component forms in terms of \(f_r\) and \(Q_i\) derivatives:

\[ \alpha_\phi=\Im[\alpha]=-2Q_i\frac{T}{f_r}\frac{\partial f_r}{\partial T}, \qquad \alpha_A=\Re[\alpha]=-\left(\frac{T}{Q_i}\frac{\partial Q_i}{\partial T}+\frac{T}{f_r}\frac{\partial f_r}{\partial T}\right), \]

\[ \beta_\phi=\Im[\beta]=-2Q_i\frac{a}{f_r}\frac{\partial f_r}{\partial a}, \qquad \beta_A=\Re[\beta]=-\left(\frac{a}{Q_i}\frac{\partial Q_i}{\partial a}+\frac{a}{f_r}\frac{\partial f_r}{\partial a}\right), \]

where \(a=\sqrt{f_rQ_iP_0}\) as used in the paper text around Eq. (4). In this project these appear as input parameters \(\alpha_A,\alpha_\phi,\beta_A,\beta_\phi\) in the Python model.

Eq. (10)-(13): Normalized Sideband-to-Demodulated Model

Eq. (10) defines voltage normalization (\(V_0=I_0R_0/2\)). Eq. (11) is the central linear system \(M\,Y=N\), where \(Y=[r,\phi,T]^T\) are demodulated normalized coordinates and \(N\) is the source/noise vector.

Eq. (12) gives the coordinate transforms from sideband variables to demodulated normalized variables. Eq. (13) expands \(M(\omega)\) explicitly in terms of \(Q, Q_i, \alpha, \beta, P_0, C, G, T_0\), and detuning.

Concise step-by-step transformation summary: Paper Transform Summary.

Interpretation for implementation: build \(M(\omega)\) directly from measured or fitted resonator parameters, then solve \(Y=M^{-1}N\) for each source and frequency bin.

Dedicated reference: M Matrix.

Model scope note: this matrix formulation is a small-signal linearization about an operating point; it is not valid for strongly nonlinear large-signal excursions, which require a nonlinear time-domain model.

Eq. (14)-(15): Time-Domain Dynamics and Pulse Model

Eq. (14) converts the frequency-domain model to time-domain dynamics \(\dot{Y}=M_tY+D_1^{-1}N\), where \(M_t\) is obtained from \(M(\omega)\).

Eq. (15) gives explicit coupled ODEs for \(r(t)\), \(\phi(t)\), and \(T(t)\) on resonance. The absorbed calorimeter power enters as an additive source term \(P(t)/C\) in the thermal equation.

Use this form to check stability (all eigenvalues of \(M_t\) must have negative real parts), pulse shape, and thermal/electrical coupling behavior before noise propagation.

Eq. (16)-(20): Noise Vectors and NEP

Eq. (16): phonon noise enters as a thermal-band power source with RMS scale \(\sqrt{4k_BT_b^2G}\).

Eq. (17): Johnson noise is represented in sideband form with separate amplitude-like and phase-like source vectors.

Eq. (18): transformed normalized Johnson vectors \(N_{J,A}\) and \(N_{J,\phi}\), including the correlated Joule-power fluctuation component in the thermal entry.

Eq. (19): excess electronic (quasiparticle generation) noise is modeled as a multiplier of the Johnson vectors, \(N_g=M_eN_J\), with \(M_e\sim\sqrt{E_{qp}/k_BT_0}\).

Eq. (20): phase-direction NEP is computed from output phase noise divided by phase responsivity to power:

\[ \mathrm{NEP}_{\phi}(\omega)=\left|\frac{[Y_{\mathrm{RMS}}(\omega)]_{\phi}}{(M^{-1})_{\phi,\,P}}\right|. \]

This is the core equation to convert modeled/measured phase noise into calorimeter-equivalent input power noise.

Python methods: Sensor.phase_responsivity_complex_rad_per_W_at_hz(), Sensor.phase_responsivity_mag_rad_per_W_at_hz(), Sensor.phase_responsivity_rad_per_W, Sensor.nep_from_phase_asd_W_per_rtHz().

Calorimeter Modeling Workflow

  • Fit \(S_{21}(f,T,P)\) to extract \(Q_i, f_r\), and derivatives needed for Eq. (4) sensitivities.
  • Construct \(M(\omega)\) with Eq. (10)-(13) and verify stable \(M_t\) from Eq. (14)-(15).
  • Build source vectors for phonon, Johnson, excess electronic, TLS, and readout noise using Eq. (16)-(19).
  • Propagate each source via \(Y=M^{-1}N\); sum output noise in quadrature (with correlations where needed).
  • Compute \(\mathrm{NEP}_{\phi}\) with Eq. (20), then feed the resulting PSD into the optimal filter on the Signal Processing page.

Notation convention used here: \(T_b\) is bath temperature and \(T_0\) is bolometer operating-point temperature under bias.

Python implementation links: Sensor.estimates(), Sensor.tau_res_s, Sensor.tau_th_s, Sensor.sphi_total_per_hz, and Sensor.nep_phi_total_W_per_rtHz.

KID Lumped Formulas

Geometric inductance estimate from trace geometry:

\[ L_g\approx \mu_0\ell\left[\ln\!\left(\frac{2\ell}{w}\right)+0.5\right]. \]

Total inductance from kinetic fraction \(\alpha_k\):

\[ L_{\mathrm{tot}}=\frac{L_g}{1-\alpha_k}. \]

Resonator capacitance and nominal impedance:

\[ C_{\mathrm{res}}=\frac{1}{(2\pi f_0)^2L_{\mathrm{tot}}},\qquad Z_0=(2\pi f_0)L_{\mathrm{tot}},\qquad R_0=\frac{Z_0}{Q_r}. \]

Python: Sensor.L_geo_H, Sensor.L_total_H, Sensor.C_res_F, Sensor.Z0_res_Ohm, Sensor.R0_Ohm.

With \(T_c\) as a basic parameter:

\[ \Delta=1.764\,k_BT_c,\qquad E_{qp}=\Delta. \]
