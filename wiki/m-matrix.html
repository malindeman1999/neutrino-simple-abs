Core Equation

The normalized small-signal system is written as:

\[ M(\omega)\,Y(\omega)=N(\omega),\qquad Y(\omega)=M^{-1}(\omega)\,N(\omega). \]

\(Y=[r,\phi,T]^T\) are demodulated normalized states and \(N\) is the source vector.

Role of \(M(\omega)\)

\(M(\omega)\) is the dynamic coupling operator that includes resonator response, thermal response, and electrothermal coupling.

  • Diagonal terms set intrinsic response poles/scales.
  • Off-diagonal terms encode coupling between \(r,\phi,T\).
  • Detuning enters through model coefficients and changes phase slope/responsivity.

Every source/noise path is computed through this same operator via \(Y=M^{-1}N\).

Implementation Form

In code, \(M(\omega)\) is built directly from project parameters \((Q_r,Q_i,\alpha,\beta,P_0,C,G,T_0,f_0,\delta f)\) and evaluated at demodulated frequency \(f\).

For this project’s nominal estimates, \(f=0\) Hz in the demodulated band is used by default.

Python: Sensor.m_matrix(), Sensor.m_matrix_array().

Full \(M(\omega)\) Matrix Form Used in Code (Eq. 13 Structure)

Let \(\omega_0=2\pi f_0\), demodulated angular frequency \(\omega\), loaded \(Q\equiv Q_r\), and dimensionless detuning \(x\equiv \delta f/f_0\). The implemented normalized matrix is:

\[ M(\omega)= \begin{bmatrix} \dfrac{2i\omega Q_i}{\omega_0}+\dfrac{Q_i}{Q}+\beta_A+4Q_iQx^2 & -\dfrac{4i\omega Q_iQx}{\omega_0} & -2Qx\beta_A+\dfrac{2\alpha_A}{T_0}\\[1.0em] \dfrac{4i\omega Q_iQx}{\omega_0}-\beta_\phi & \dfrac{2i\omega Q_i}{\omega_0}+\dfrac{Q_i}{Q}+4Q_iQx^2+2Qx\beta_\phi & -\dfrac{2\alpha_\phi}{T_0}\\[1.0em] -\left(1+\dfrac{\beta_A}{2}\right)P_0 & Qx(\beta_A+2)P_0 & i\omega C+G-\dfrac{P_0\alpha_A}{T_0} \end{bmatrix}. \]

This is the exact matrix currently implemented in Sensor.m_matrix().

Connection to \(M_t\) and Stability

Time-domain linear dynamics are obtained from \(M(\omega)\) using:

\[ D_0=M(0),\qquad D_1=-i\,\frac{\partial M}{\partial\omega},\qquad M_t=-D_1^{-1}D_0. \]

See Mt Stability for eigenvalue checks and stability criteria.

Scope

This matrix model is a small-signal linearization about an operating point. Large-signal nonlinear behavior requires a nonlinear time-domain model.
