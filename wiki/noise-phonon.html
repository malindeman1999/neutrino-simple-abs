Phonon Noise Vector

Eq. (16) source term and mapping into normalized coordinates.

HomeTheoryNoise SourcesPhononSignal Processing

Physical Expression

Thermal-link fluctuation power RMS:

\[ P_{ph,\mathrm{RMS}}=\sqrt{4k_B T_0^2 G\,F_{\mathrm{link}}(T_0,T_b,n)}. \]

\[ F_{\mathrm{link}}=\frac{n+1}{2n+3}\, \frac{1-(T_b/T_0)^{2n+3}}{1-(T_b/T_0)^{n+1}}. \]

Default in this project: \(n=3\) for SiN-like legs.

This source is purely thermal-band power injection.

Noise Vector Derivation

In sideband/source notation, phonon fluctuations do not directly drive USB/LSB voltage terms, only the thermal source component.

So the source vector is:

\[ N_{ph}=\begin{bmatrix} 0\\0\\\sqrt{4k_B T_0^2 G\,F_{\mathrm{link}}(T_0,T_b,n)} \end{bmatrix}. \]

This is Eq. (16), already in the normalized model coordinate system used with \\(Y=M^{-1}N\\).

Vector Component Units

Noise vectors are mixed-unit objects in the \((r,\phi,T)\) basis, so units are component-wise:

\[ \mathrm{units}(N_{ph})= \begin{bmatrix} 1/\sqrt{\mathrm{Hz}}\\ 1/\sqrt{\mathrm{Hz}}\\ \mathrm{W}/\sqrt{\mathrm{Hz}} \end{bmatrix}. \]

In this model, phonon source is introduced directly in normalized coordinates (no separate sideband \(n\)-vector stage).

Propagation and NEP

Output from this source alone:

\[Y_{ph}=M^{-1}N_{ph}.\]

Phase-direction contribution:

\[\mathrm{NEP}_{\phi,ph}(\omega)=\left|\frac{[Y_{ph}(\omega)]_{\phi}}{(M^{-1})_{\phi,\,P}}\right|.\]
