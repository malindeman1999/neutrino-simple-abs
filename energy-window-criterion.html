Energy Truncation from Finite Pulse Window

Exponential-pulse tail truncation criterion for target energy-error floor.

Home Signal Processing Pileup Analysis Energy Window Criterion

Assumption

For stationary noise and a single-pole exponential pulse shape:

\[ p(t)\propto e^{-t/\tau}, \qquad t\ge 0. \]

If energy is estimated by finite-window area integration up to time \(T\), the fractional missed area is:

\[ \frac{\Delta E}{E}=e^{-T/\tau}. \]

1 Part in 20000 Criterion

Set target truncation error:

\[ \frac{\Delta E}{E}\le \frac{1}{20000}=5\times10^{-5}. \]

Then:

\[ e^{-T/\tau}\le\frac{1}{20000} \Rightarrow \frac{T}{\tau}\ge\ln(20000)\approx 9.90. \]

Result: need about \(10\tau\) of pulse window for this worst-case truncation criterion.

Use in Pileup Logic

This sets a consistent pileup-overlap criterion: treat overlap as potentially biasing if a second event begins before \(9.90\tau\) of the first pulse has elapsed.

Linked application page: Pileup Analysis.

Note: this is conservative; model-based pulse fitting can reduce effective bias below this worst-case direct-integration bound.
